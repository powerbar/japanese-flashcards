# Japanese Flashcards

A simple flashcard app for learning Japanese vocabulary. 
Final project for Javascript 1 Class at Mission College, Fall 2025 by Hsiu-hsien Chen

## Features
- 100 Japanese vocabulary words (from Chapter 1 to 4 of Genki, 3rd Edition, published by the Japan Times Publishing)
- Search functionality
- Random card selection
- Track known words
- Hiragana, Kanji, and English display

## How to Use
1. Visit: https://powerbar.github.io/japanese-flashcards/
2. Load the vocabulary.txt file
3. Start studying!

## File Format
vocabulary.txt format:
```
hiragana,kanji,english
きのう,昨日,yesterday
```
